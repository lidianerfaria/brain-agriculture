## Instalação:
1. **Clone o repositório:** <br/>
`git clone https://github.com/lidianerfaria/brain-agriculture.git`

2. **Instale as dependencias:** <br/>
`yarn`

3. **Rode a API:** <br/>
`yarn server`

4. **Rode o projeto:** <br/>
`yarn run dev`
